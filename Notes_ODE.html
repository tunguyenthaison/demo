Abstract

The classic Existence results

Global Existence Theorem for Lipschitz right hand sides
Let $I$ be an interval of the form $I = [a,b)$ or $I = [a,b]$, and let $f:I\times \mathbb{R}^n\to \mathbb{R}^n$ be a function that is uniformly Lipschitz in the space variable, i.e., there is an $L\in \mathbb{R}$ such that \begin{equation*} \vert (t,x) - f(t,y)\vert \leq L|x-y|\qquad \forall\; t\in I, x,y\in \mathbb{R}^n. \end{equation*} The the initial value problem \begin{equation} \begin{cases} x'(t) = f(t,x(t)), \qquad t\in I,\\ x(a) = \underline{x} \end{cases} \tag{IVP} \end{equation} has a unique solution for every $x_0\in \mathbb{R}^n$.
Let us consider $I = [a,b]$ is bounded. Using the integral form of (IVP), we will seek for solution as a fixed point of an integral operator. Let $\lambda>0$ be chosen later, we define $E = \left\{\varphi\in \mathrm{C}([a,b];\mathbb{R}^n):e^{-\lambda t}|\varphi(t)|\in L^\infty([a,b])\right\}$ with norm \begin{equation*} \Vert \varphi\Vert_{E} = \sup_{t\in [a,b]} e^{-\lambda t}|\varphi(t)|. \end{equation*} It is clear that $\left(E,\Vert \cdot \Vert_E\right)$ is a Banach space. We define a map $\mathcal{F}:E\to E$ by \begin{equation*} (\mathcal{F}\varphi)(t):= \underline{x} + \int_a^t f(s,\varphi(s))ds, \qquad \varphi\in E. \end{equation*} For $\textbf{x}(\cdot),\textbf{y}(\cdot) \in E$ one has \begin{equation*} \left\Vert\mathcal{F}\textbf{x}(t) - \mathcal{F}\textbf{y}(t)\right\Vert_E \leq \frac{L}{\lambda}\Vert \textbf{x} - \textbf{y}\Vert_E. \end{equation*} If $\lambda > L$ then $\mathcal{F}$ is a contraction. The Banach fixed point theorem implies that $\mathcal{F}$ has unique fixed point $\textbf{x}$ such that $\Vert \textbf{x} - \textbf{x}_0 \Vert_E < r $ where $\textbf{x}_0(t) \equiv \underline{x}$ for $t\in [a,b]$ where \begin{equation*} \sup_{t\in [a,b]} e^{-\lambda t}\left\vert \int_a^t f\left(s,\underline{x}\right)ds \right\vert = \left\Vert \mathcal{F}\textbf{x}_0 -\textbf{x}_0 \right\Vert_E \leq \left(1-\frac{L}{\lambda}\right)r. \end{equation*} Since $[a,b]$ is compact and $f$ is continuous, such a choice of $r$ exists, thus we obtain the existence of solution for the case $I$ is compact. If $I = [a,b)$, we have a unique solution $\textbf{x}_T:[a,T]\to \mathbb{R}^n$, thus we can define $\textbf{x}:I\to \mathbb{R}^n$ by setting $\textbf{x}(t) = \textbf{x}_T(t)$ if $t\leq T < b$. The uniqueness of $\textbf{x}_T$ implies that any other solution $\tilde{\textbf{x}}:I\to \mathbb{R}^n$ must coincide with $\textbf{x}$ on every interval $[a,T]$, thus must coincide with $\textbf{x}$ everywhere in $I$.
